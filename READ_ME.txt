Μέσα στον φάκελο my_data περιέχονται οι εξής κατάλογοι:
--> HDFS_data: Κατάλογος που περιέχει οτιδήποτε αποθηκεύεται μέσα στο HDFS κατά την διάρκεια εκτέλεσης της εφαρμογής.
--> Kafka_data: Κατάλογος που περιέχει οτιδήποτε αποθηκεύεται μέσα στο Kafka κατά την διάρκεια εκτέλεσης της εφαρμογής.

Μέσα στον φάκελο my_documentation υπάρχει η αναφορά της εργασίας (E17065.pdf).

Μέσα στον φάκελο my_files περιέχονται οι εξής κατάλογοι:
--> Avro_schema: Κατάλογος που περιέχει το AVRO Schema που κατασκευάζει η εφαρμογή κατά την εκτέλεσή της.
--> dynamic_web_project: Κατάλογος που περιέχει όλο το java dynamic web project που κατασκευάστηκε (ISG_final)
--> Flume_configuration_file: Κατάλογος που περιέχει .conf αρχείο που κατασκευάζει η εφαρμογή κατά την εκτέλεσή της.
